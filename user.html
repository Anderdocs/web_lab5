Симетричне Шифрування

Симетричне шифрування — це метод шифрування, при якому для шифрування та дешифрування даних використовується один і той самий секретний ключ. Основні особливості:
1) Швидке й ефективне.
2) Вимагає безпечної передачі ключа між сторонами.
3) Приклади алгоритмів: AES, DES, RC4.
Використовується для забезпечення конфіденційності даних у закритих системах або там, де можна захистити ключ.
